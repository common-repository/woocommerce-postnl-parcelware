This plugin exports all selected orders to PostNL Parcelware Desktop. - NO SUPPORT
== Description ==
***Quickly export selected orders to PostNL Parcelware.***

_This plugin is not maintained anymore_

Because of changes at PostNL we can no longer develop this plugin or give support for it. SORRY guys!

== Installation ==

1. Install WooCommerce Postnl ParcelWare either via the Wordpress.org plugin directory or by uploading the files to the '/wp-content/plugins/' directory.
2. Activate the plugin  through the 'Plugins' menu in WordPress.

== Upgrade Notice ==
Please backup first.
== Usage ==

Go to WooCommerce > Parcelware, configure and start exporting. Simple!
